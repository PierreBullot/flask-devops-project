### Créer la structure de tests  

Créer un dossier tests/ à la racine du projet   
Créer le fichier tests/test_app.py pour y regrouper les tests.  
Créer un fichier conftest.py vide à la racine de votre projet pour éviter les erreurs d'import !  
Rajouter pytest à requirements.txt
